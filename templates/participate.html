Contribute

Everyone can contribute to the Monero network in some manner.
  • Run a node: Host a node on the Monero network.
  • Mining: Contribute hash power to the Monero network and secure transactions.
  • Development: Have some coding skills? Help develop the Monero project.
  • Submit an aboutmonero article: Add a page to the aboutmonero website.

Communities

Join in the discussion! These are various community groups across the web participating in the Monero project.
  • Reddit: A public forum for discussing Monero and asking questions.
  • Stack Exchange: Ask questions and receive answers from Core developers and other members of the community.
  • BitcoinTalk Thread: A public forum to discuss the Monero project.
  • Telegram Chat: Public chatroom for discussing Monero.
  • Source Code: The github repository for the Monero code.
  • Monero Research Lab: The leading research group advancing monero's privacy technology.
  • Monero Talk Youtube Channel: A regular video series interviewing and discussing Monero related topics.
  • Monero Community Workgroup Youtube Channel: Contains many videos for learning about the monero project.
